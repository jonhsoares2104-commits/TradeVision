 📈 Analisador de Ações em Python

Um projeto desenvolvido em Python para analisar ações do mercado financeiro de forma simples e automatizada.

O sistema permite buscar dados históricos, gerar gráficos, calcular indicadores e ajudar na análise de ativos.

---

 🚀 Funcionalidades

✅ Buscar dados históricos de ações  
✅ Exibir gráfico de preço  
✅ Calcular média móvel  
✅ Mostrar variação percentual  
✅ Exibir volume negociado  
✅ Gerar relatório automático  
✅ Interface simples no terminal

---

 🛠 Tecnologias usadas

- Python 
- pandas
- matplotlib
- yfinance
